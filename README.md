# Dheeraj Jha - Personal Portfolio

## Overview
A professional portfolio website showcasing my work as a UI Engineer and Software Developer.

## Technologies Used
- React
- Vite
- Tailwind CSS
- Framer Motion

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
